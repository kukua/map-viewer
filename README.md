# Map Viewer

> Simple map viewer for specific measurement data.

### Usage

```bash
git clone https://github.com/kukua/map-viewer.git
cd map-viewer
git checkout quick-demo
cp .env.example .env
chmod 600 .env
# > Edit .env

docker-compose up -d
```

Tested with NodeJS v5.1
